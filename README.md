# Gemini Clone

A clone of Google's Gemini AI chat interface built with React and the Gemini API.

## Features

- Chat with Google's Gemini AI
- Typing effect on AI responses
- Recent chat history in the sidebar
- New chat button to reset the conversation
- Suggestion cards on the home screen

## Built With

- [React](https://react.dev)
- [Vite](https://vite.dev)
- [Google Generative AI SDK](https://www.npmjs.com/package/@google/generative-ai)
